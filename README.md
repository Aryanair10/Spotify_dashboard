# 🎧 Spotify_dashboard
This interactive Power BI dashboard analyzes Spotify track data to explore streaming trends, artist popularity, and audio features. The dashboard provides insights into track characteristics such as danceability, energy, valence, speechiness, and acousticness, along with top tracks, popular artists, and stream distribution over time.

Key features include:

-Total Streams KPI showing overall platform streaming volume.

-Audio Feature Analysis using gauges for danceability, energy, and valence.

-Top Tracks by Streams highlighting the most streamed songs.

-Popular Artists by Track Streams to identify leading artists.

-Track Characteristics Comparison through scatter plots.

-Streaming Trends Over Time to observe growth in music consumption.

-Interactive Filters (key, artist name, released year, mode) for dynamic exploration.

-This dashboard helps users identify music trends, analyze song attributes, and understand streaming patterns across artists and years.
